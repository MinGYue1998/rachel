## 二、域名解析配置（阿里云控制台）

1. 登录 [阿里云域名控制台](https://dc.console.aliyun.com/)
2. 找到域名 `toumakazusa.fun`，点击「解析」
3. 添加以下记录：

| 记录类型 | 主机记录 | 记录值 | TTL |
|----------|----------|--------|-----|
| A | @ | 服务器公网IP | 600 |
| A | www | 服务器公网IP | 600 |

## 三、服务器环境搭建

连接服务器后执行：

```bash
# 更新系统
apt update && apt upgrade -y

# 安装基础工具
apt install -y curl wget git vim

# 安装 MySQL 8.0
apt install -y mysql-server

# 安装 Python 3.11
apt install -y python3.11 python3.11-venv python3-pip

# 安装 Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 安装 Nginx
apt install -y nginx

# 安装 pnpm
npm install -g pnpm

# 安装 certbot（用于申请免费SSL证书）
apt install -y certbot python3-certbot-nginx
```

## 四、配置 MySQL

```bash
# 启动 MySQL
systemctl start mysql
systemctl enable mysql

# 安全配置（按提示设置root密码，其他选项选Y）
mysql_secure_installation

# 登录 MySQL（输入刚才设置的root密码）
mysql -u root -p

# 执行以下SQL（替换 your_password 为你的密码）
CREATE DATABASE rachel DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'rachel'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON rachel.* TO 'rachel'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

## 七、部署后端

```bash
cd /opt/vue-fastapi-admin

# 创建虚拟环境
python3.11 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 创建环境变量文件（替换密码）
cat > .env << 'EOF'
DB_HOST=localhost
DB_PORT=3306
DB_USER=rachel
DB_PASSWORD=your_password
DB_DATABASE=rachel
EOF

# 创建 systemd 服务
cat > /etc/systemd/system/fastapi.service << 'EOF'
[Unit]
Description=Vue FastAPI Admin
After=network.target mysql.service

[Service]
Type=simple
User=root
WorkingDirectory=/opt/vue-fastapi-admin
EnvironmentFile=/opt/vue-fastapi-admin/.env
ExecStart=/opt/vue-fastapi-admin/venv/bin/python run.py
Restart=always
RestartSec=3

[Install]
WantedBy=multi-user.target
EOF

# 启动后端
systemctl daemon-reload
systemctl enable fastapi
systemctl start fastapi

# 检查状态
systemctl status fastapi
```

## 九、配置 Nginx（绑定域名）

```bash
# 创建 Nginx 配置
cat > /etc/nginx/sites-available/vue-fastapi-admin << 'EOF'
server {
    listen 80;
    server_name toumakazusa.fun www.toumakazusa.fun;

    # 前端静态文件
    location / {
        root /opt/vue-fastapi-admin/web/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api {
        proxy_pass http://127.0.0.1:9999;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
EOF

# 启用配置
ln -s /etc/nginx/sites-available/vue-fastapi-admin /etc/nginx/sites-enabled/
rm /etc/nginx/sites-enabled/default

# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx
```

## 十、配置 HTTPS（免费SSL证书）

```bash
# 申请证书（自动配置 Nginx）
certbot --nginx -d toumakazusa.fun -d www.toumakazusa.fun

# 按提示输入邮箱，同意条款
# 选择是否重定向 HTTP 到 HTTPS（建议选 2-重定向）

# 设置自动续期
systemctl enable certbot.timer
systemctl start certbot.timer
```

## 十一、阿里云安全组配置

在阿里云控制台添加安全组规则：

| 端口 | 协议 | 授权对象 | 说明 |
|------|------|----------|------|
| 22 | TCP | 0.0.0.0/0 | SSH |
| 80 | TCP | 0.0.0.0/0 | HTTP |
| 443 | TCP | 0.0.0.0/0 | HTTPS |
| 3306 | TCP | 127.0.0.1 | MySQL（仅本地） |
